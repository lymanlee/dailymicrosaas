## 技术难度

**技术栈参考**

- 前端：Canvas API 或 WebAssembly（纯客户端处理性能更好）
- 后端：Sharp / Jimp / Pillow，或者接入推理 API
- 客户端处理优先：能在浏览器里做的，不要传到服务器，用户会更放心
