# 详情页整改方案

## 问题清单与解决方案

### 1. 竞品分析排版优化

**当前问题**
- 三列卡片布局过于拥挤
- 信息密度高，阅读疲劳

**整改方案**
- 改为横向滑动卡片或垂直列表
- 增加卡片间距和留白
- 竞品信息分层展示：名称/价格 + 弱点（弱化显示）
- 可选：收起为「查看 X 个竞品」点击展开

---

### 2. 评分星星样式优化

**当前问题**
- 未满格的星星使用黑色填充，视觉突兀

**整改方案**
- 改为描边空心样式（stroke-only）
- 或改为灰色填充（slate-300）
- 保持已填充星星为黄色/琥珀色

**示意**
```
当前: ★★★★★ (第5个黑色)
改为: ★★★★☆ (第5个空心/灰色)
```

---

### 3. 深度拆解箭头问题

**当前问题**
- 同时出现向下和向上两个箭头，逻辑矛盾

**整改方案**
- 只保留一个箭头，根据展开状态切换
- 收起时显示「向下箭头 + 展开」
- 展开时显示「向上箭头 + 收起」
- 移除文字标签中的箭头，仅图标变化

---

### 4. 深度拆解内容去重

**当前问题**
- 展开后与页面上方内容大量重复

**整改方案**

**方案 A：精简为「补充细节」**
- 移除已展示的内容（痛点、竞品、变现等）
- 保留只有深度拆解才有的内容：
  - 技术实现细节
  - 具体用户访谈引用
  - 数据背后的分析逻辑
  - 失败案例/踩坑经验

**方案 B：改为「原始数据」展示**
- 展示原始调研数据（HN 帖子、Reddit 讨论等）
- 让用户看到一手证据

**方案 C：移除折叠，直接精简页面**
- 如果内容确实重复，考虑直接删除折叠区块
- 将独有内容分散到对应模块下方

**推荐：方案 A**，保留有价值的补充信息，删除重复内容。

---

## 实施优先级

1. **P0** - 星星样式（快速修复，视觉问题）
2. **P0** - 箭头修复（交互问题）
3. **P1** - 竞品排版（布局优化）
4. **P1** - 深度拆解去重（内容优化）
