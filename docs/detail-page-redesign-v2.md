# 详情页 Redesign V2 方案

> **文档状态**：部分已实现（2026-04-12 更新）
>
> ✅ 已实现：删除深度拆解模块、侧栏精简、视觉统一。
>
> ⏳ 未实现：五维度评分（含 TAM）、五维雷达图、竞品动态加载（需先完成 `src/data/competitors/` 链路）。

---

## 核心改动

### 1. 删除深度拆解模块
- 将技术实现细节 → 并入「切入策略」模块
- 将用户调研洞察 → 并入「用户痛点」模块  
- 将验证策略 → 并入「MVP 路线图」模块

### 2. 机会评分增加 TAM 维度
```
新五维度：
- 需求强度 (Demand)
- 竞争空间 (Competition)
- 进入门槛 (Barrier)
- 变现潜力 (Monetization)
- 市场规模 (TAM) ← 新增
```

### 3. 删除侧栏「关键数据」板块
- 保留：机会速览、快速行动、订阅表单
- 移除：搜索趋势、竞品数、验证周期、TAM 数字卡片

### 4. 整体视觉升级

#### 4.1 信息层级优化
```
Hero 区域（高视觉权重）
├── 标题 + 分类标签
├── 一句话价值主张
├── 机会评分五维雷达/卡片
└── 核心结论徽章

主内容流（中等密度）
├── 需求证据（信号卡片）
├── 用户痛点（图标+描述）
├── 市场格局（简化竞品列表）
├── 切入策略（含技术细节）
├── 变现路径（价格锚点）
├── MVP 路线图（时间轴）
└── 风险提醒（琥珀色卡片）

侧栏（轻量辅助）
├── 机会速览（五维评分）
├── 快速行动（复制/分享）
└── 订阅 CTA
```

#### 4.2 视觉风格统一
- **圆角系统**：统一使用 16px/24px 圆角
- **间距系统**：8px 基准，模块间 32px/48px
- **色彩系统**：
  - 主背景：白色卡片 + slate-50 页面底
  - 强调色：amber-500（评分）、emerald-500（正向）、sky-500（信息）
  - 风险色：amber-600（替代红色）
- **字体层级**：
  - 标题：font-bold tracking-tight
  - 正文：text-slate-600 leading-relaxed
  - 标签：uppercase tracking-wider text-xs

#### 4.3 组件精修
- **评分展示**：五维进度条 → 或改为雷达图/星形图
- **痛点卡片**：图标 + 标题 + 描述，减少色块使用
- **竞品列表**：极简行式布局，悬停显示详情
- **MVP 路线图**：垂直时间轴，阶段清晰标注
- **风险卡片**：图标 + 风险 + 缓解策略，紧凑排列

#### 4.4 动效与交互
- 模块入场：stagger fade-in
- 悬停反馈： subtle lift + shadow
- 展开/收起：smooth height transition
